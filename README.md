# Flower Shop Website

An online flower and gift shop website developed for educational purposes.  
The system supports product browsing, ordering, and basic management features.

## Technologies Used

-   PHP (MVC Architecture)
-   MySQL
-   HTML / CSS / JavaScript
-   Bootstrap
-   XAMPP
-   Git & GitHub

## Main Features

### User Features

-   View product list
-   View product details
-   Add products to cart
-   Place orders
-   Track order status

### Admin Features

-   Manage products
-   Manage categories
-   Manage orders
-   Manage users
-   Manage promotions

## Installation Guide

### 1. Clone the repository

git clone https://github.com/ngdinhthuan04/flower_shop.git

### 2. Move project to XAMPP

Copy the project folder into:
xampp/htdocs/

### 3. Database setup

-   Open phpMyAdmin
-   Create a database named flower_shop
-   Import the provided .sql file from the database folder
